# visual-organizer

A NeuroDock skill that turns prose, meeting notes, or a stream-of-consciousness dump into a Mermaid diagram. Choose a flowchart for a process, a sequence diagram for actor interactions over time, or a mind map for loosely related concepts.

## Calling it

- `/visualize` or `/diagram` as an explicit slash command.
- Or say "draw this out" / "make a diagram" in plain prose.

The diagram always sits alongside a prose answer, never replaces it. Output includes a plain-language accessible description so screen-reader users get the same content. The project palette is single-hue neutral; motion is suppressed when the user's profile sets `preferences.motion: reduced`.

## Lineage

Builds on the pattern and approach pioneered by [`/neurodivergent-visual-org`](https://github.com//neurodivergent-visual-org). Credit, with thanks.
