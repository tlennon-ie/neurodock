# audhd-context-recovery

A `/resume` command. When you open a session after time away — overnight, a weekend, two weeks — and the first question is "where was I?", this skill answers it. It reads the local cognitive graph and reconstructs your last active project, your three most recent decisions, and any open threads (blockers and follow-ups). It then suggests one concrete next move, drawn from the graph.

The skill never invents facts. If the graph is thin, it says so plainly.

It activates on `/resume`, `/where-was-i`, the phrases "where was I" / "what was I working on", or on session-open when more than eight hours have passed since the last prompt (in which case it asks before running, rather than running automatically).

It is AuDHD-targeted because the combination of ADHD context-loss and autistic deep-context-investment makes reconstruction especially costly. The skill carries the cost so the user does not have to.
