# NeuroDock papers

Long-form pieces from the project: position papers, methods notes,
and field-study write-ups. Each paper is versioned, numbered, and
published in lockstep with an arXiv submission and a post on
`neurodock.org/papers`.

## How papers move

A paper here is a draft until the council marks it `accepted` in
this index and an arXiv identifier is recorded. The bibliography
for each paper lives next to it as a `.bib` file. Edits to an
accepted paper open a new revision (e.g. `0001-v2.md`) rather than
overwriting; arXiv versioning is the source of truth.

The council is consulted on every draft before submission. The
clinical advisory board is consulted on any paper that names
guardrail behaviour, detection heuristics, or clinical framing.
See `GOVERNANCE.md` and `ETHICS.md` at the repo root.
