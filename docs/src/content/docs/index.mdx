## What you can do today

All three pillars are live on `main` and installable from npm + PyPI. See [`CHANGELOG.md`](https://github.com/tlennon-ie/neurodock/blob/main/CHANGELOG.md) for the per-package version index.

### Substrate

Five MCP servers, all published to PyPI:

- [`mcp-chronometric`](/reference/mcp-servers/chronometric/) `v0.0.2` ‚Äî time, session, and break management.
- [`mcp-cognitive-graph`](/reference/mcp-servers/cognitive-graph/) `v0.0.3` ‚Äî persistent entity memory and recall, local-first, with a 4-rung resolution cascade (exact ‚Üí alias ‚Üí fuzzy ‚Üí embedding).
- [`mcp-task-fractionator`](/reference/mcp-servers/task-fractionator/) `v0.0.3` ‚Äî decompose vague goals into atomic 5‚Äì90 minute tasks (ISO 8601 duration parsing).
- [`mcp-translation`](/reference/mcp-servers/translation/) `v0.0.2` ‚Äî deterministic-baseline-plus-LLM-envelope tools for incoming subtext, outgoing tone, and meeting briefs.
- [`mcp-guardrail`](/reference/mcp-servers/guardrail/) `v0.0.3` ‚Äî rumination, hyperfocus, and sycophancy detectors. All three live.

Six skills compose those servers into daily-use workflows:

- [`adhd-daily-planner`](/reference/skills/adhd-daily-planner/) ‚Äî Monday-morning brief.
- [`audhd-context-recovery`](/reference/skills/audhd-context-recovery/) ‚Äî `/resume` from yesterday's state.
- [`ocd-decision-finalizer`](/reference/skills/ocd-decision-finalizer/) ‚Äî refuse to re-validate a closed decision.
- [`hyperfocus-formatter`](/reference/skills/hyperfocus-formatter/) ‚Äî "Answer First" output shaping.
- [`visual-organizer`](/reference/skills/visual-organizer/) ‚Äî Mermaid for overwhelm states.
- [`asd-meeting-translator`](/reference/skills/asd-meeting-translator/) ‚Äî transcript ‚Üí four-section brief with verbatim ambiguity quotes.

### Communication layer

`mcp-translation` `v0.0.2` ships a deterministic baseline plus an LLM refinement envelope, both behind one MCP contract. Bidirectional translation of corporate communication: literal-translate incoming subtext, tone-check outgoing messages, structure meeting transcripts. The browser extension (`@neurodock/extension-browser` `v0.0.4`) wires five real LLM providers ‚Äî Ollama (local default), LM Studio, Anthropic, OpenAI, and OpenRouter (including `openrouter/auto`) ‚Äî plus a deterministic mock lane for development. Non-localhost local providers (e.g. LM Studio bound to a LAN address) are gated behind a per-host `chrome.permissions` runtime grant.

### Guardrails

`mcp-guardrail` `v0.0.3` ships all three detectors live: rumination, hyperfocus, and sycophancy (48 tests). Transparent, user-configured, overridable. Thresholds live in the user's profile under `guardrails`.

## Principles

- Lower friction for users **and contributors**.
- Local-first by default. Cloud is opt-in per scope.
- The user is the authority. Self-ID sufficient.
- Composable over monolithic.
- Refuse where appropriate.

Read the full [manifesto](/manifesto/) for what these mean in practice.
