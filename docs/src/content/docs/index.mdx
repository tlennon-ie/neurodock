NeuroDock consolidates the currently fragmented ecosystem of ADHD, autistic, AuDHD, and OCD-supportive AI tooling into one composable stack. It is anchored by three pillars: a **substrate** of skills and MCP servers that externalise executive function, a **communication layer** that translates corporate ambiguity and modulates tone, and a **clinical guardrails** layer that prevents AI from amplifying rumination, hyperfocus, and sycophancy.

## What you can do today (v0.2.1)

All three pillars are live on `main` and installable from npm + PyPI. See [`CHANGELOG.md`](https://github.com/tlennon-ie/neurodock/blob/main/CHANGELOG.md) for the per-package version index.

### Substrate

Three MCP servers, all published to PyPI:

- [`mcp-chronometric`](/reference/mcp-servers/chronometric/) `v0.0.1` — time, session, and break management.
- [`mcp-cognitive-graph`](/reference/mcp-servers/cognitive-graph/) `v0.0.2` — persistent entity memory and recall, local-first, with a 4-rung resolution cascade (exact → alias → fuzzy → embedding).
- [`mcp-task-fractionator`](/reference/mcp-servers/task-fractionator/) `v0.0.2` — decompose vague goals into atomic 5–90 minute tasks (ISO 8601 duration parsing).

Six skills compose those servers into daily-use workflows:

- [`adhd-daily-planner`](/reference/skills/adhd-daily-planner/) — Monday-morning brief.
- [`audhd-context-recovery`](/reference/skills/audhd-context-recovery/) — `/resume` from yesterday's state.
- [`ocd-decision-finalizer`](/reference/skills/ocd-decision-finalizer/) — refuse to re-validate a closed decision.
- [`hyperfocus-formatter`](/reference/skills/hyperfocus-formatter/) — "Answer First" output shaping.
- [`visual-organizer`](/reference/skills/visual-organizer/) — Mermaid for overwhelm states.
- `asd-meeting-translator` — transcript → four-section brief with verbatim ambiguity quotes.

### Communication layer

`mcp-translation` `v0.0.1` ships a deterministic baseline plus an LLM refinement envelope, both behind one MCP contract. Bidirectional translation of corporate communication: literal-translate incoming subtext, tone-check outgoing messages, structure meeting transcripts. Real provider wiring (Ollama, Anthropic, OpenAI) is on the [Q3 2026 roadmap](https://github.com/tlennon-ie/neurodock/blob/main/ROADMAP.md#next--q3-2026).

### Guardrails

`mcp-guardrail` `v0.0.2` ships all three detectors live: rumination, hyperfocus, and sycophancy (48 tests). Transparent, user-configured, overridable. Thresholds live in the user's profile under `guardrails`.

## Principles

- Lower friction for users **and contributors**.
- Local-first by default. Cloud is opt-in per scope.
- The user is the authority. Self-ID sufficient.
- Composable over monolithic.
- Refuse where appropriate.

Read the full [manifesto](/manifesto/) for what these mean in practice.
